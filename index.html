    GarrysModServerSetup

    This will help you get your Debian server working. You can install Virtualbox if you have Windows or some other operating system.

    Requirements

    You're gonna need an OK computer. Here's the recommended computer specs for GMod:

    OS: WindowsÂ® 7/8/8.1/10 (You can use Debian for the server.) Processor: 2.5 GHz Processor or better Memory: 8 GB RAM Graphics: 1GB dedicated VRAM or better DirectX: Version 9.0c Network: Broadband Internet connection Storage: 10 GB available space

    Note, It is NOT recommended to play on the server and host the server on the same PC.

    Network

    This is the part for the NETWORKING. You're going to need to forward the port 27015 through 27005, both TCP and UDP. This is pretty much it for networking. You might want to turn off your firewall, too.

    Getting the script

    You will have to install Virtualbox with Debian Jessie for this to work if you are not a debian user. Set it to 4880 MB, and set the amount of cores.
