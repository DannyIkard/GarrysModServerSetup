GarrysModServerSetup

This will help you get your Debian server working. You can install Virtualbox if you have Windows or some other operating system.

Requirements

You're gonna need an OK computer. Here's the recommended computer specs for GMod:

OS: WindowsÂ® 7/8/8.1/10 (You can use Debian for the server.) Processor: 2.5 GHz Processor or better Memory: 8 GB RAM Graphics: 1GB dedicated VRAM or better DirectX: Version 9.0c Network: Broadband Internet connection Storage: 10 GB available space

Note, It is NOT recommended to play on the server and host the server on the same PC.

Network

This is the part for the NETWORKING. You're going to need to forward the port 27015 through 27005, both TCP and UDP. This is pretty much it for networking. You might want to turn off your firewall, too.

Getting the script

You will have to install Virtualbox with Debian Jessie for this to work if you are not a debian user. Set it to 4880 MB, and set the amount of cores.

Download the script, and extract the .ZIP, then open the folder. Launch the SetupGarrysModServer.sh script. It's very self-explanatory, so I'm going to skip the part.

After...

So, there should be a GMOD_ReadMe.txt. Read it, and it should tell you how to turn it on. Lets say you turn on GMod, and go look for your server, you join, but it's laggy. I had this issue, but I put in something to make it work. Go to /home/gm/Steam/steamapps/common/GarrysModDS/garrysmod/cfg/server.cfg (Directory may vary) and put in (anywhere) the following:

sv_minrate 0 sv_maxrate 0 sv_minupdaterate 10 sv_maxupdaterate 66

It might not work for everyone, but it worked for me.

More Server.cfg

You can customize your server with the server.cfg. Here's a good website to help! http://robinschoors.com/?p=244 Copy the text into your server.cfg.

Mounting CSS

It will install CSS:Dedicated server for you. But, to mount it, follow these steps. Go to /home/gm/Steam/steamapps/common/GarrysModDS/garrysmod/cfg/mount.cfg Open it, and you will see something like this:

"mountcfg" { //"cstrike" "C:\steamcmd\steamapps\common\Counter-Strike Source Dedicated Server\cstrike" //"tf" "C:\mytf2server\tf" }

First, find the //"cstrike", and remove the 2 //'s. then, replace the directory (The C:\steamcmd\blahblah in the quotes) with "/home/gm/Steam/steamapps/common/Counter-Strike Source Dedicated Server/cstrike/" (Directory may vary), save, and restart server.

That would be about everything, you can contact me on steam or something if you have issues. http://steamcommunity.com/id/Senvr11/
